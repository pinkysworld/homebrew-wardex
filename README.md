# homebrew-wardex

Homebrew tap for Wardex.

## Install

```bash
brew tap pinkysworld/wardex
brew install wardex
```

## Formula

The tap publishes `Formula/wardex.rb`, which installs released Wardex binaries from:

- https://github.com/pinkysworld/Wardex/releases

The formula is synced automatically from published Wardex releases by the release automation in `pinkysworld/Wardex`.
